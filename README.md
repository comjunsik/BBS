# jsp를 이용한 게시판

jsp와 js, css, bootstrap을 이용하여 간단한 게시판 기능을 구현해 보았습니다.
회원가입, 로그인, 게지판 조회, 글쓰기, 수정, 삭제 등의 기능을 연습해 보았습니다.
